# Instrucciones para obtener la URL de conexión directa de Supabase

1. Ve a: https://supabase.com/dashboard/project/thwbgjszzimirthpukty/settings/database

2. Busca la sección "Connection string"

3. Selecciona el modo "Session mode" o "Direct connection" (NO uses "Transaction mode")

4. Copia la URL que tiene el puerto **5432** (NO 6543)

5. La URL debería verse así:
   postgresql://postgres.thwbgjszzimirthpukty:[TU_PASSWORD]@aws-0-us-west-2.pooler.supabase.com:5432/postgres

6. Reemplaza [TU_PASSWORD] con: Rastro@2228

7. Pega la URL completa aquí cuando la tengas
